Resources

This page links to information about the datasets that are available as part of the McKnight Brain Aging Registry.

Procedures in MBAR: This Manual of Procedures details the procedures that were performed with all registry participants. That document describes what happened on each day of testing.

Data elements available: The Data List documents the types of data that were collected as part of each of these procedures. Note that the list is organized based on the Manual of Procedures, so that if there is a question about how a particular type of data was collected, you can query the other pages for more information, including the Manual of Procedures as well as the Details about each of the assessments including forms used and handbooks for assessments.

The files within the Data Dictionary document the specific data elements that are included in tabular data. Note that this website does not contain the data itself, you must request access to obtain data.

How to access data: Members of the MBRF research Institutes can access the data for testing hypotheses for new publications or preliminary data for grant proposals. To do so, complete this Data Access Form , including information about which datasets you plan to use in your analyses (using language as used in the Data List and Data Dictionary). Email this Data Access form to pstew@uab.edu with Subject line “MBAR data access form.” The MBAR Scientific Advisory Committee meets monthly to review these data access forms. After review, and scientific advisory committee approval, our team will collate the datasets needed for the proposal to be shared with you on the Box platform.

Note: There are just over 200 participants’ data in this dataset, and thousands of data elements per participant. Not all participants have data for each of the data elements. For example, some participants moved too much in the scanner to acquire their BOLD fMRI, and so do not have that data. If you would like to know the number of participants who have usable data for a given set of data elements, please email pstew@uab.edu and our team will work with you to find out how many data points are available.

Types of data: These datasets involve many types of data. Keep in mind that Raw data will need to be processed. In particular, frozen blood samples are stored at our University of Florida site, and in order to analyze and process those data, you will need to thaw it. Because blood samples are finite, we will combine proposals which require access to blood in order minimize the number of times that samples must be thawed.

  • Raw data: after basic quality control and checking, as obtained from participants. These raw data include:
    • Raw MRI datasets
    • Blood; frozen blood samples. (These are limited in quantity)
    • Trial-level scores on individual behavioral and neuropsychological tests
  • Minimally preprocessed MRI data: this involves applying a set of standard preprocessing algorithms to the data, to prepare for a set of typical data analyses. For example, skull stripping, identification of white matter and pial boundaries, identification of timepoints of movement in functional MRI, etc.
  • Processed images: whole brain image files; one per participant, obtained after data processing. For example, individual participants’ whole brain functional connectivity to the posterior cingulate cortex.
  • Tabular data: This involves data that has been processed to give a single value for each participant. These data include, for example:
    • Participant demographics
    • Behavioral data factor scores indicating weighting on each of 5 factors
    • FMRI data indicating strength of connection among brain regions designated as the “Default Mode Network”
    • Data that are potentially identifiable (e.g., age over 89 years old) will be distributed only to investigators who are on the IRB for this project.
